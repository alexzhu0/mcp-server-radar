# Changelog

## v0.2.0 - 2026-05-31

- Promoted the project to a trend-aligned flagship MCP utility.
- Added registry/list input shapes: `servers`, `items`, `repositories`, and `mcpServers`.

- Added URL/auth/description-based risk inference and per-server risk scores.

- Added server rows with transport, auth, and risk categories.
- Added Markdown table output.
- Kept capability index output stable.

- Initial public release of `mcp-server-radar`.
- Included README, examples, tests, community files, and social preview source.
