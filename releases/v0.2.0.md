# mcp-server-radar v0.2.0

Trend-aligned release for MCP server discovery and agent tool selection.

## Added

- Registry and awesome-list style input support through `items`, `repositories`, and `mcpServers`.
- Capability inference from tags, categories, keywords, and descriptions.
- Risk score per server based on auth, network, filesystem, write, delete, admin, and secret surfaces.
- Markdown table output tuned for README and issue comments.

## Try It

```bash
PYTHONPATH=src python3 -m mcp_server_radar examples/registry-items.json --format markdown
PYTHONPATH=src python3 -m mcp_server_radar examples/registry-items.json --format json
```
